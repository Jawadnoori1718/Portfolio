# Project images

Drop each project's screenshot here using the **exact filename** below. The site
loads `/projects/<name>.<ext>` and accepts `.png`, `.jpg` or `.webp` (it tries
them in that order). Until a file exists, the card shows the generated SVG art as
a fallback — nothing breaks.

## Filenames

- `drift`       → Drift (Global Analytics Platform)
- `streetpulse` → StreetPulse (Urban Safety dashboard)
- `hillingone`  → HillingOne (Smart Booking System)
- `robot`       → Robot Shape Drawing System

Example: save the Drift screenshot as `public/projects/drift.png`.

## For best results

- The image area is **16:9** and shown with `object-fit: cover`, so use roughly
  16:9 (landscape) images — anything close works; the edges may crop slightly.
- ~1200×675 px is plenty.
